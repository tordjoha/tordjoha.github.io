Most used commands

General Vim

  • y copy text
  • p paste text
  • d cut

Plugins

  • :Goyo distraction free mode
  • :FZF fuzzy finder
  • :Git ... for git commands in Vim

Vim Wiki

  • space + ww open default wiki index
  • space + wt open default wiki index in new tab
  • space + ws select and open wiki index file
  • space + wd delete wiki file youre in
  • Enter follow link
  • Backspace go back link
  • Tab tab through links
  • ShiftTab go back through links
  • :VimwikiAll2HTML convent all wiki links to HTML
  • :Vimwiki2HTML convert current to HTML
  • _ .. _ for italics
  • * .. * for bold
