The await language

  • Declarations as usual (int x = 5)
  • Sequential statements: if (condition) statements; while;...
  • Arrays: a[n]
  • For loops: for [i = 0 to n]...
  • Concurrent statements: co statement1; || ...; || statementN; oc
  • Processes:
    • Process A {...}
    • Process B [i = 1 to N] {...}
    • A process declaration is essentially an abbreviation for a co statement with one arm and/or one quantifier.
    • Process declaration occurs at the syntactic level of a procecure declaration.
    • Informal: process ~thread.

The Await statement

<await (B) S;> B - Boolean condition (delay condition) S - a list of statements

<await (x == y) z = a + b; w = c + d; >

  • Atomic actions are specified by means of the angle brackets <..>
  • The await statement can be used to specify arbitrary coarse-grained atomic

actions.

  • The general form of the await statement specifies both mutual exclusion and

condition synchronization.

  • To specify only mutual exclusion we abbriviate the await statement: <S;>
  • To specify only condition synchronization, we abbriviate the await statement:

<await(B);>

  • Example: Delaying the executing process until count > 0; <await (count>0);>
